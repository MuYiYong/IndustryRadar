# IndustryRadar

IndustryRadar 是一个面向图数据库、数据库、开源技术、AI、大模型、Agent、金融科技、本体、数据治理、公司与竞品动态的行业观察仓库。

仓库会持续归档每日行业简报、每周趋势总结，以及围绕重点主题、公司、论文和市场动态沉淀的知识库内容。

## 如何每天收到更新提醒

如果你希望每天收到本仓库的更新邮件，可以在 GitHub 上关注这个仓库：

1. 点击仓库右上角的 **Star**，收藏并支持这个项目。
2. 点击右上角的 **Watch**。
3. 在下拉菜单中选择 **All Activity**。

选择 **All Activity** 后，只要仓库有新的简报、报告或知识库更新，GitHub 就会根据你的通知设置发送提醒邮件。

> 如果没有收到邮件，请检查 GitHub 的通知设置：
> **Settings → Notifications → Email notifications**，确认邮件通知已开启。

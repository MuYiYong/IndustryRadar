# 悦数｜Yueshu

## 2026-07-20｜W29 战略追加

### 行业机会判断

企业 AI 正从模型与 GraphRAG 竞争转向 Agent 的语义、权限、执行和证据基础设施。悦数可将图数据库定位从“复杂关系查询与分析引擎”升级为“企业 Agent 的关系与语义基础设施”。

### 建议产品叙事

> 悦数图不仅存储和查询关系，更为企业 Agent 提供语义理解、关系推理、权限判断、行动验证和证据追踪。

### 建议能力组合

1. AI 入口：AskGraph、GraphRAG、MCP、自然语言生成图查询与图算法；
2. 关系推理：多跳查询、路径分析、图算法、风险与影响传播；
3. 企业安全：SSO、证书认证、细粒度权限、审计、密钥和多租户；
4. 生产能力：高可用、备份恢复、滚动升级、资源管理和可观测性；
5. 语义治理：本体、规则、血缘、证据链、Dry Run 和行动约束。

### 优先差异化方向

- Agent Control Graph：统一建模 Agent、用户、工具、数据、规则、动作和影响；
- Ontology-driven Tools：由本体生成受约束工具和参数接口；
- Agent Dry Run：执行前模拟权限、规则、后续动作和业务影响。
